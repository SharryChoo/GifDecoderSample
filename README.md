## GifDecoderSample
Core is GIFLIB

## 功能描述
- 使用 GIFLIB 进行 Native 层的 GIF 解码
- 提供了获取 GIF 帧的降采样操作
- 使用 FrameSequenceDrawable 提供了基于双缓冲机制的 GIF 动画播放
- 替换了 Glide 的 GIF 加载
